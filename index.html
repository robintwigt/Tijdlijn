Tijdlijn Webdesign

HTML

HTML staat voor ‘Hyper Tekst Markup Language’. Door middel van HTML kun je de inhoud van webpagina’s structureren. Je kunt met HTML er dus voor zorgen dat er informatie te vinden is op jouw website, je kunt echter niet de vormgeving van de elementen aanpassen aan jouw wensen.

Tabellen

Tabellen worden gebruikt om informatie te structureren. Sommige websites werden volledig gemaakt van een tabel, zodat een website makkelijker in vakken ingedeeld kon worden. Hierdoor konden dingen verticaal uitgelijnd worden en hierdoor werd er op de website gebruik gemaakt van een soort van grid.

Javascript

Met HTML kon je bepaalde dingen niet bereiken, daarom kwam er Javascript. Als je bijvoorbeeld een pop-upvenster wilt, kan je dit niet maken met HTML, hiervoor is Javascript nodig. Met Javascript kunnen dynamische webpagina’s gemaakt worden. Tegenwoordig willen we Javascript liever vermijden als hetzelfde bereikt kan worden door CSS te gebruiken.

Flash

Door middel van Flash kan je webinhoud maken waarin je geluid, video en animatie kunt gebruiken. Flash is gebaseerd op vectoren, hierdoor is de bestandsgrootte klein bij eenvoudige vormen en is in- en uitzoomen mogelijk. Flash was eerst van Marcomedia en werd later overgenomen door Adobe en is nu dus onderdeel geworden van hun softwareprogramma.

CSS

Met CSS kun je een pagina vormgeven. De informatie die op de website moet komen te staan, staat in een HTML-bestand en die informatie kun je dus weergeven zoals jij dat wilt door gebruik te maken van CSS. Het duurde een paar jaar voordat de browsers CSS volledig ondersteunden en er zaten veel bugs in. Hoe de website eruit zag hing af van je browser.

Mobiel Design

Mobiel design was erg lastig en er kwamen veel problemen bij kijken. Designers vroegen zich bijvoorbeeld af of ze het design op de mobiel hetzelfde moesten maken als het design op de computer. Voor mobiel design maken de designers gebruik van het 960 grid systeem en de 12 kolommen divisie.

Responsive Web Design

Responsive webdesign is een website die zich aanpast aan het formaat van het scherm waarop de website bekeken wordt. Als de website bekeken wordt op de computer ziet het er dus anders uit dan wanneer de website op een mobiel wordt weergegeven. Er wordt nog steeds gebruik gemaakt van HTML en CSS, alleen werk je bijvoorbeeld met percentages in plaats van pixels.

Flexbox

Flexbox maakt onderdeel uit van CSS. Met flexbox kan je een layout creëren en wijzigen op een manier die je erg veel tijd zal besparen. Het is namelijk niet nodig om veel code te schrijven, maar je kunt gebruik maken van een bepaalde eigenschap. Door gebruik te maken van flexbox is je website responsive. De website past zich dus aan aan het formaat van het scherm.
